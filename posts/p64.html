Visualization software for “Pendent de Vot” theatre performance

Feb 29, 2012 • ad_min

PdV github CODE

Roger Bernat page

Pending of Votation is a theatre performance by Roger Bernat. It has been represented in several countries and it uses a visualization and live data management software developed by Jaume Nualart and Mar Canet.

The piece has some complexity because it deals with more than 100 live polls from the audience and the show has no actors.

The software reads votes from a voting hardware and pass results to a python manager that finelly send the results to an OpenFrameworks visualizer.

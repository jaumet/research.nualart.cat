Visualization tool: CentredNets

Sep 11, 2013 • ad_min

Sample 1: stenophylla

Sample 2: salicina

Sample 3: two maps

This a visualization tool to represent networks centered in nodes. Instead of edges this approach uses node size  when a node is clicked.

Three examples can be seen with a sample of genetic ecology. Colors indicate same category. When click a node, the network resize nodes acording to the one clicked. Mouse-over shows the name of each node.

The data for this prototype was a contribution with Dr Francisco Encinas-Viso (CSIRO, Australia)

code: JS, RaphaelJS
